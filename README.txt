##Smart Traffic Light Management System with Visual Interface
##Project Overview
This project simulates a smart traffic light system that dynamically adjusts signal timing based on vehicle density. It uses C++ for logic and HTML/CSS/JavaScript for visual representation.

##Objectives
- Simulate real-life traffic signals
- Reduce waiting time
- Demonstrate smart traffic concepts
- Provide visual interface

##Technologies Used
   C++, HTML, CSS, JavaScript
##Features
- Dynamic signals
- Vehicle-based timing
- Console + UI simulation
- Signal transitions

##Working Principle

Vehicles determine timing:
<10 → 5 sec
10–19 → 10 sec
20+ → 15 sec

##How to Run
Compile: g++ main.cpp -o traffic
Run: traffic

Conclusion
This project demonstrates intelligent traffic control using simple logic.
